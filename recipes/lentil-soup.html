Lentil Soup

Description

Lentil soup is a cheap, easy, and delicious meal. It's perfect for any time you want something filling and nutritious, but you don't want to spend too much cash.

This recipe is great, but what's truly awesome about this dish is its versatility. Don't like these ingredients? Dont use them! Have some stuff you haven't been able to use but don't want to waste? Toss it in! Pretty much any veggie or herb you like will add depth and nutrition to this feast-on-a-budget.
